# DFIR Scripting Toolkit

This repository contains Python scripts developed for Digital Forensics and
Incident Response (DFIR) tasks.

The goal of this project is to automate common log analysis and investigation
steps that are typically performed during incident response.

## Repository Structure

- `scripts/`  
  Contains individual DFIR tools. Each tool has its own folder and documentation.

- `data/`  
  Contains sample log files used for testing and analysis.

## Skills Demonstrated

- Python scripting for DFIR
- Log parsing and analysis
- Dictionary-based frequency analysis
- Clean repository structure
- Evidence-oriented thinking

## Disclaimer

All scripts are for educational and defensive security purposes only.
